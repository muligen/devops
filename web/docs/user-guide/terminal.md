# Agent 终端使用指南

Agent 终端是 AgentTeams 系统提供的聊天式命令交互界面，让运维人员能够快速执行命令并实时查看结果。

## 功能概览

- 🚀 快速执行命令
- 📊 实时查看输出
- 📝 命令历史记录
- ⚙️ 快捷命令按钮
- 🎨 简洁直观的界面

## 开始使用

### 1. 打开终端

1. 访问左侧菜单的「Agent」页面
2. 点击任意一个 Agent 进入详情页
3. 点击「终端」标签页

### 2. 执行命令

#### 方式一：手动输入

1. 在底部输入框中输入命令（例如：`ls -la`）
2. 按 `Enter` 键执行命令
3. 结果会实时显示在上方消息区域

#### 方式二：使用快捷命令

1. 点击左侧的快捷命令按钮
2. 命令会自动填充并执行
3. 右键点击快捷命令可以只填充不执行

### 3. 查看输出

- **正在执行**：显示「执行中...」状态和动画
- **执行完成**：显示退出码和执行时长
- **执行失败**：显示红色错误消息

## 高级功能

### 命令历史

使用方向键浏览历史命令：

- `↑` 上箭头：查看上一条命令
- `↓` 下箭头：查看下一条命令

历史记录按 Agent 分类存储，即使刷新页面也会保留。

### 多行命令

如需执行多行命令：

1. 输入第一行命令
2. 按 `Shift + Enter` 换行
3. 继续输入后续命令
4. 按 `Enter` 执行完整命令

### 输出控制

- **复制输出**：点击结果消息右上角的「复制」按钮
- **展开更多**：当输出较长时，点击「展开完整输出」查看全部内容
- **清空终端**：点击终端顶部的「清空」按钮清除显示的消息

### 常用快捷命令

| 名称 | 命令 | 说明 |
|------|------|------|
| 列出文件 | `ls -la` | 列出当前目录详细信息 |
| 查看进程 | `top -b -n 1` | 显示进程信息（静态） |
| 查看磁盘 | `df -h` | 显示磁盘使用情况 |
| 查看内存 | `free -h` | 显示内存使用情况 |
| 查看网络 | `netstat -ano` | 显示网络连接 |
| Ping 测试 | `ping -n 4 8.8.8.8` | 测试网络连通性 |

### 添加自定义快捷命令

1. 查找需要添加的命令
2. 通过终端首次执行该命令
3. 系统会自动记录并方便后续使用

## 界面说明

### 顶部状态栏

显示当前 Agent 的连接状态和执行任务数：

- 🟢 **在线**：Agent 正常连接，可以执行命令
- ⚫ **离线**：Agent 断开连接，命令输入框将被禁用

### 消息区域

- **蓝色气泡**：用户输入的命令
- **灰色/绿色气泡**：Agent 返回的执行结果
- **红色气泡**：错误消息

### 输入区域

底部固定显示的命令输入框，支持多行输入和换行。

## 注意事项

1. **Agent 状态**：Agent 必须在线才能执行命令
2. **命令限制**：单条命令最长 4096 字符
3. **历史限制**：最多保存 100 条历史命令
4. **独立存储**：每个 Agent 的命令历史独立存储

## 常见问题

**Q: 为什么无法执行命令？**

A: 请检查 Agent 是否在线。如果agent离线，命令输入框会被禁用。

**Q: 命令执行后没有输出？**

A: 部分命令可能需要较长时间执行，请耐心等待「执行中...」状态完成。

**Q: 如何查看完整的执行结果？**

A: 如果输出内容超过显示阈值，会出现「展开完整输出」按钮，点击即可。

**Q: 命令历史会保存多久？**

A: 命令历史存储在浏览器 localStorage 中，永久保存直到手动清除浏览器数据。

**Q: 可以同时操作多个 Agent 吗？**

A: 可以在不同 的 Agent 终端之间切换，每个 Agent 都有独立的历史记录。
